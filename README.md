# Mi Tienda de Calzados GAMARRA
Proyecto Tienda de Calzados V1

# Objetivo del Proyecto
Simular la gestión básica de una tienda de calzados. Implementar funcionalidades como pedido de compras, categoria de calzados. Probar el uso de C# en aplicaciones y formularios orientadas a objetos.

# Descripción
El presente proyecto tiene como finalidad simular la gestión básica de una tienda de calzado, incorporando funcionalidades esenciales como la administración de pedidos de compra y la clasificación de los productos por categorías.

# Tecnologías utilizadas
Lenguaje: C#, Entorno de desarrollo: Visual Studio 22, Base de Datos: SQL Server 2022
